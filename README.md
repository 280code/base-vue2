# base-vue2

## 简单介绍
本模版集成了vue2，用于进行主体开发，集成了element并且进行全局引入，方便使用，接着使用了vue-router进行路由管理，最后使用vuex进行状态管理，使用了axios进行前后端交互。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
